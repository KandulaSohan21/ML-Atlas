# Linear Regression

Linear Regression is a supervised machine learning algorithm used to predict continuous numerical values.

It establishes a linear relationship between one or more independent variables and a dependent variable.

The goal is to find the best-fit line that minimizes prediction errors.

## Formula

y = mx + c

Where:

- y = Predicted value
- x = Input feature
- m = Slope of the line
- c = Intercept

## Learning Type

Supervised Learning

## Problem Type

Regression
