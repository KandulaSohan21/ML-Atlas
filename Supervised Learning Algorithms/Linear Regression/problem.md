# Problem Statement

An automobile manufacturer is designing new vehicle models and wants to estimate the amount of CO₂ emissions a vehicle will produce before it is released to the market.

Conducting physical emission tests for every design variation is expensive and time-consuming. The company already has historical data containing vehicle specifications such as engine size, number of cylinders, and fuel consumption ratings.

Using this historical data, the company wants to build a machine learning model that can predict CO₂ emissions for new vehicle designs.

## Business Objective

Reduce testing costs and accelerate vehicle development by accurately estimating CO₂ emissions using vehicle specifications.

## Input Features

* Engine Size (L)
* Cylinders
* Fuel Consumption City (L/100 km)
* Fuel Consumption Highway (L/100 km)
* Fuel Consumption Combined (L/100 km)

## Target Variable

* CO₂ Emissions (g/km)

## Success Criteria

Predict vehicle CO₂ emissions with reasonable accuracy before physical testing is performed.

## Why This Problem Matters

Governments worldwide enforce strict emission standards. Manufacturers that exceed these limits may face regulatory penalties, increased taxes, or restrictions on vehicle sales.

An accurate prediction model helps engineers design more environmentally friendly vehicles while reducing development costs.
